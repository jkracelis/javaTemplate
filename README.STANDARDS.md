## 1. Naming Conventions
  - Classes / Interfaces / Enums
    - PascalCase
    - File name must match class name
    - Methods
  - camelCase
    - Start with a verb
  - Variables
    - camelCase
    - Descriptive names
  - Constants
    - UPPER_CASE_WITH_UNDERSCORES
    - Must be static final

## Formatting
